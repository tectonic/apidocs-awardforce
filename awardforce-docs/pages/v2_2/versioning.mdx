---
title: "Versioning"
description: "Award Force uses semantic versioning with major and minor versions."
---

- **MAJOR** version: Incremented for incompatible changes.
- **MINOR** version: Incremented for minor breaking changes.

Use the following `Accept` header values for version 2.2:

```bash
application/vnd.Creative Force.v2.2+json
application/vnd.Creative Force.v2.2+xml
```

To use an earlier API version, replace `2.2` with the required version number.
