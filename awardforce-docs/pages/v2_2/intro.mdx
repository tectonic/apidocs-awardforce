## Welcome to the Award Force API

**API version:** 2.2

Use the Award Force API to integrate Award Force with websites, applications, and internal systems.
Use the API to exchange data between Award Force and external platforms.

## Available resources

Use the API to list, retrieve, create, update, delete, and upload data across the following resource groups:

<AccordionGroup>
  <Accordion title="Core resources">
    - Account
    - Season
    - User
    - Webhook
  </Accordion>
  <Accordion title="Entry management">
    - Entry
    - Entry (realtime)
    - Attachment
    - Field
    - File
    - Contributor
  </Accordion>
  <Accordion title="Judging and evaluation">
    - Assignment
    - Review task
    - Round
    - Score set
  </Accordion>
  <Accordion title="Organisation">
    - Category
    - Chapter
    - Leaderboard
  </Accordion>
  <Accordion title="Commerce">
    - Order
    - Tax
  </Accordion>
</AccordionGroup>
