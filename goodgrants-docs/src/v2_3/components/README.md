Reusable components
===========

- You can create the following folders here:
  - `schemas` - reusable [Schema Objects](https://spec.openapis.org/oas/v3.2.0.html#schema-object)
  - `responses` - reusable [Response Objects](https://spec.openapis.org/oas/v3.2.0.html#response-object)
  - `parameters` - reusable [Parameter Objects](https://spec.openapis.org/oas/v3.2.0.html#parameter-object)
  - `examples` - reusable [Example Objects](https://spec.openapis.org/oas/v3.2.0.html#example-object)
  - `headers` - reusable [Header Objects](https://spec.openapis.org/oas/v3.2.0.html#header-object)
  - `requestBodies` - reusable [Request Body Objects](https://spec.openapis.org/oas/v3.2.0.html#request-body-object)
  - `links` - reusable [Link Objects](https://spec.openapis.org/oas/v3.2.0.html#link-object)
  - `callbacks` - reusable [Callback Objects](https://spec.openapis.org/oas/v3.2.0.html#callback-object)
  - `securitySchemes` - reusable [Security Scheme Objects](https://spec.openapis.org/oas/v3.2.0.html#security-scheme-object)
- Filename of files inside the folders represent component name, i.e. `Customer.yaml`
