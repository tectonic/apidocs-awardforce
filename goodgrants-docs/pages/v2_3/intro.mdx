## Welcome to the Good Grants API

**API version:** 2.3

Use the Good Grants API to integrate Good Grants with websites, applications, and internal systems.
Use the API to exchange data between Good Grants and external platforms.

## Available resources

Use the API to list, retrieve, create, update, delete, and upload data across the following resource groups:

<AccordionGroup>
  <Accordion title="Core resources">
    - Account
    - Season
    - User
    - Webhook
  </Accordion>
  <Accordion title="Application management">
    - Application
    - Application (realtime)
    - Attachment
    - Field
    - File
    - Contributor
  </Accordion>
  <Accordion title="Review and evaluation">
    - Assignment
    - Action task
    - Round
    - Score set
  </Accordion>
  <Accordion title="Organisation">
    - Category
    - Chapter
    - Leaderboard
  </Accordion>
  <Accordion title="Commerce">
    - Order
    - Tax
  </Accordion>
</AccordionGroup>
